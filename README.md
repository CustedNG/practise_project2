## 长春天气APP

## 快速开始

**1.下载代码到本地**

```
git clone https://github.com/CustedNG/practise_project2.git
```

**2.使用pub拉起依赖**

```
flutter pub get
```

**3.使用flutter run运行**

```
flutter run
```
